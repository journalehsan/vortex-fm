Get System-Wide Theme Sync

Install Vortex FM

git clone https://github.com/journalehsan/vortex-fm.git
cd vortex-fm
cargo build --release

Enable System Sync

// Automatic system-wide sync
vortex --enable-system-themes

// All Cosmic apps will follow

System-Wide Sync Active

Once installed, Vortex FM automatically synchronizes themes with all Cosmic applications through the Omarchy theme system.
